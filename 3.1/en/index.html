Getting Started



Step 1
Search Zenbo Scratch App at Launcher
Step 2
Click Zenbo Scratch App
Step 3
After entered app, Zenbo will tell your wifi IP and SSID. We need these information to connect ot Scratch afterwards.
Step 4
Please confirm the upper-right control permission switch is ON! Now we finished steps at Zenbo Junior, next we start to set connection to Scratch!
Step 5
Please download Scratch 3.0. You can click here to download:
Step 6
After downloaded and unzip the file, please open the file ZenboScratch.html. We suggest you to use Chrome browser to open it.
Step 7
After Scratch opened, we can start to build programs!
Step 8
Before start programming, remember to run the "set IP" block, and then you can use other blocks to control Zenbo Junior!
Step 9
If your program only control single Zenbo Junior, you can use the single-device extensions.
Step 10
If your program will control multi Zenbo Juniors, you can use the multi-device extensions.
